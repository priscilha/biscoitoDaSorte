# biscoitoDaSorte

Biscoito da Sorte, um jogo no qual o usuário, a partir de um clique ou enter, abre um biscoito com a sua sorte do dia.


- Estrutura de dados HTML
- Animações com CSS
- Funções no Javascript
- Manipulação da DOM
- Biblioteca JS Math()
- Funções *callback*
- Arrays
